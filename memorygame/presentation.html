About the Project

This is a fun and interactive web-based memory matching game built using HTML, CSS, and JavaScript. The objective of the game is to match pairs of cards by flipping them over, while tracking the time and number of moves taken to complete the game.

  • Features: Timer, Moves Counter, High Score Tracker
  • Technologies Used: HTML, CSS, JavaScript
  • Visuals: Emoji-based card faces for added fun!
